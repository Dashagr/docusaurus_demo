Checkout creation and configuration

After registering your account, you can create a checkout for accepting payments. The checkout in our system is a collection of accounts (wallets or purses) to which payers' funds will be received.

To one checkout can be linked a few purses in different currencies, which you are using during payers invoicing. For example, if you issue an invoice in USD, the wallet at the selected checkout will be topped up with the same currency after payment.

Funds are received in the currency in which the invoice was created, but the payer can pay using any payment method available for this checkout. For example, if the checkout currency is USD, and the customer pays for the product in EUR, the amount will be converted and credited to the checkout in USD. You can see the current currency exchange rate from this URL: https://api.interkassa.com/v1/currency.

For the convenience of traffic management and data monitoring, you can create several checkouts for one website.

Create a checkout​

Step 1. Log in to your account.​

Log in to your account from the log in page.

Step 2. Fill the form​

On the main screen, click on the "Create a checkout" button and fill out the form with the following information:

  • Checkout type: Personal (if the checkout is needed for standard receiving payments) or Agent (for participation in the referral program).
  • Site URL: The URL of the site that will accept online payments.
  • Checkout name: whatever you like. If there are several checkouts, it will be easier to find them by name.
  • Checkout Currency: UAH (additional currencies can be included later in the checkout settings. After these currencies are enabled with your account manager).

Checkout creation

Step 3. Click the 'Create' button.​

Congratulations! The checkout has been created. Now you need to confirm it to activate payment methods.

Checkout verification​

Step 1. Enter the verification page​

In the 'My Checkouts' section, click on 'Verify checkout'.

Website ownership confirmation

Step 2. Confirm the website ownership​

You can confirm ownership in several ways:

  • Copy the meta tag generated by Interkasa and insert it into the HTML code of the main page of the site inside the <head> </head> tag.

Meta tag implementation

OR

  • Download an attached txt file and place it in the root folder of your website.

Step 3. Verify the site​

Save the changes, return to Interkassa and click the 'Verify' button.

The verification page

Congratulations! The checkout has been created and confirmed. Now you can start generating the payment form.
